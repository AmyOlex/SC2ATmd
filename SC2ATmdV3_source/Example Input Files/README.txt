Description of example input files.

FOM/Clustering format type
300geneTCexpt1.txt : Used in FOM and both clustering walk throughs.
300geneTCexpt2.txt : same list of genes as expt1 above, but data is from a replicate experiment.  Can use in the two-file consensus clustering.

Heatmap/Stats format type
300geneTCexpt1_heatstats.txt : Used in Heatmap and Statistics Generation walk-through.

Cluster Mapping format type
300geneTCexpt1-mappingInput.txt : an example input file for cluster mapping.

Custom CLuster Run Import for Consensus Clustering
300geneTCexpt1_CustomImport.txt : an example custom cluster solution import file for consensus clustering analysis.
